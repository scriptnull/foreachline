CONTRIBUTE

foreachline is a open source project developed and maintained via Github

You can contribute to it in 3 ways.

  • Command Line Tool - Build something that can make the command line tool more useful and go live in npm.
  • gh-pages - The online version of foreachline is built with Angular.js. To the best , I have added some material design in it. However , it will be complete when it has Animations in it.
  • Snippets - If you got a snippet that you use with foreachline often. Share it with the community by creating a gist and adding it here

If you have any requests , doubts , issues or any other thing that you want to discuss about foreachline , come lets have a chat in gitter
