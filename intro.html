Introduction

This is an online version of foreachline, which is available as command line tool via npm

What it does ?

You input 2 things source and logic.

Lets say your source looks like this ,


Apple
Orange
Banana

By typing in JavaScript code in logic , you can get something like this ,


I love Apple
I love Orange
I love Banana

or like this ,


<li>Apple</li>
<li>Orange</li>
<li>Banana</li>

or like this ,


1.Apple
2.Orange
3.Banana

or like this ,


{"id":1,"name":"Apple"}
{"id":2,"name":"Orange"}
{"id":3,"name":"Banana"}

In most cases , converting your lines to json seems effective , as you can do manythings like generating Excel sheets with that data.

foreachline is a very simple tool. The core idea is not building something complex, rather build something simple that can solve very simple problems faced everyday.

For me , one such problem was writing code in bulk which had few patterns that can be made use of.

For example , I had about 50 variables equating to other variables , which later are equated vice versa.

So , source was ,


textBox1.value = obj.Property1; textBox2.value = obj.Property2; textBox3.value = obj.Property3;

and my expected result was


obj.Property1 = textBox1.value; obj.Property2 = textBox2.value; obj.Property3 = textBox3.value;

The logic for this piece is ,


var variables = line.replace(';','').split('='); line = variables[1] + ' = ' + variables[0] + ';' ;

I face this citeria one in five times a day , while writing code and in need of some standard reusable snippets.So , I have tried to added up snippets.If possible I request you to contribute snippets that can solve yours and someothers problem.
