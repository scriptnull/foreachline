Introduction

This is an online version of foreachline, which is available as command line tool via npm

What it does ?

You input 2 things source and logic.

Lets say your source looks like this , 

Apple
Orange
Banana

By typing in JavaScript code in logic , you can get something like this , 

I love Apple
I love Orange
I love Banana
or like this , 
<li>Apple</li>
<li>Orange</li>
<li>Banana</li>
or like this , 
1.Apple
2.Orange
3.Banana
or like this , 
{"id":1,"name":"Apple"}
{"id":2,"name":"Orange"}
{"id":3,"name":"Banana"}
